# Open problems

Open problems explicitly stated in the literature. Each entry must cite a
paper that articulates the problem; you may not invent open problems here.

- **Problem** (one sentence).
- **Stated by**: [Paper](../02_Literature/Papers/<id>.md) -- section.
- **Adjacent work**: 0-3 papers that have attacked it so far.
- **Status**: open / partially addressed / apparently solved by &lt;paper&gt;.

If you want to add a problem statement that is *your own* formulation, use
a workspace instead.
